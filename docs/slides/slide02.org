* 一般の確率測度
# 有限試行ではない一般の標本空間を考えた場合に，
# 確率測度 \(P\) がどんな性質をもたなくてはいけないかを説明する．

** 可測集合
- 可測集合 (measurable set) ::
  確率測度 \(P\) で確率の値を測ることができる集合\\
  (標本点の集合の中で事象として考えてよいもの)
- 可測 (measurable) ::
  "集合 \(A\) が可測"とは
  集合 \(A\) を確率測度 \(P\) で測ることができること

** 集合族
- 確率測度 \(P\) で確率の値を測ることができる
  対象全体 (関数 \(P\) の *定義域*) を
  \(\mathcal{F}\) と書く
- 定義域 \(\mathcal{F}\) は
  可測な事象(標本空間の部分集合) \(A\subset\Omega\) の集まり
- このような"集合の集合"を
  *集合族 (family of sets)* という

** \(\sigma\)-加法族
- 定義
  #+begin_quote
  以下の (\(\sigma.1\))-(\(\sigma.3\)) 
  の3つの条件を満たす集合族を
  *\(\sigma\)-加法族 (\(\sigma\)-algebra)*
  と呼ぶ．
  #+begin_src latex
    \begin{align}
      (\sigma.1)\quad
      &\Omega\in\mathcal{F}\\
      (\sigma.2)\quad
      &A\in\mathcal{F}
	\Rightarrow A^c\in\mathcal{F}\\
      (\sigma.3)\quad
      &A_n\in\mathcal{F},\; (n=1,2,\dotsc)
	\Rightarrow \bigcup_{n=1}^\infty A_n\in\mathcal{F}
    \end{align}
  #+end_src
  #+end_quote
  - (\(\sigma.1\))-(\(\sigma.3\)) は
    定義域 \(\mathcal{F}\) の満たすべき条件でもある
  
** 条件の意味
- (\(\sigma.1\)) ::
  標本空間(全事象)は可測である( \(P\) で測ることができる)
- (\(\sigma.2\)) :: 
  ある事象が可測なら，その余事象も可測である
- (\(\sigma.3\)) :: 
  可測集合の可算(自然数で番号付けできる)無限和も可測である

** 確率測度
- 定義
  #+begin_quote
  集合関数 \(P\) は条件(P.1), (P.2)を満たすとき
  *測度 (measure)*
  と呼ばれ，
  さらに(P.3)まで満たすとき
  *確率測度 (probability measure)*
  と呼ばれる．
  また \(P(A)\) を \(A\) の
  *測度*
  という．
  #+begin_src latex
    \begin{align}
      (P.1)\quad
      &P(A)\ge 0,\; A\in\mathcal{F}\\
      (P.2)\quad
      &P\biggl(\sum_{n=1}^\infty A_n\biggr)
	=\sum_{n=1}^\infty P(A_n),\; A_n\in\mathcal{F}\\
      (P.3)\quad
      &P(\Omega)=1
    \end{align}
  #+end_src
  #+end_quote
  - (P.2)を 
    *\(\sigma\)-加法性 (\(\sigma\)-additivity)*
    という

** 確率空間
- 定義
  #+begin_quote
  標本空間 \(\Omega\) と確率測度 \(P\) ，
  および \(P\) の定義域である 
  \(\sigma\)-加法族
  \(\mathcal{F}\) の組 
  \((\Omega,\mathcal{F},P)\) を
  *確率空間 (probability spcae)*
  という．
  #+end_quote

** ルーレット回しの確率測度
- 区間 \((0,1]\) からの無作為抽出 
  #+begin_quote
  試行 \(T\) を
  "区間 \((0,1]\) から無作為に一点抜き出すこと"
  とする．
  このとき標本空間は
  #+begin_src latex
    \begin{equation}
      \Omega=(0,1]
    \end{equation}
  #+end_src
  であり無限試行となる．
  確率測度は事象 \(A\) が
  区間 \([a,b]\), \((a,b)\), \([a,b)\) または \((a,b]\)
  といった簡単な集合であれば
  #+begin_src latex
    \begin{equation}
      P(A)=|A|=b-a,\;\text{(\(|A|\)は区間の長さを表す)}
    \end{equation}
  #+end_src
  とすればよい．
  #+end_quote

** 確率測度の定義域
- 確率測度 \(P\) の定義域 \(\mathcal{F}\) は
  集合演算で作られる
- *可算個* の区間を組み合わせてつくられる集合
  #+begin_src latex
    \begin{align}
      \mathcal{F}
      &=\{\text{可算個の任意の区間の和，差，交，}\\
      &\qquad\qquad\qquad\text{余集合から作られる集合}\}
    \end{align}
  #+end_src
- 集合の測度(確率)は (P.2) の性質より
  排反な区間の長さの和から計算できる

** Lebesgue 測度
- 定義
  #+begin_quote
  "区間 \((0,1]\) から無作為に一点抜き出す"試行 \(T\)
  によって考えられる確率測度 \(P\) を \((0,1]\) 上の
  *Lebesgue 測度 (Lebesgue measure)*
  という．
  また Lebesgue 測度の定義域となる
  \(\sigma\)-加法族を
  \(\mathbb{R}\) の
  *Borel 集合族 (Borel field)*
  という．
  #+end_quote
  - Lebesgue 測度は \(\mu\) で表されることが多い
  - 厳密な定義については Lebesgue 積分の本を参照

** Lebesgue 測度の計算
- 一点が抜き出される確率
  #+begin_quote
  Lebesgue 測度において，
  \(P\{a\}\)
  (一点 \(a\) が抜き出される確率)は \(0\) となる．

  例えば \(\{1,1/2,1/3,\dotsc\}\) という可算集合を考えると，
  #+begin_src latex
    \begin{equation}
      P\left\{\frac{1}{n}\right\}\ge\varepsilon>0
      \Rightarrow 
      P\left(\left\{1,\frac{1}{2},\frac{1}{3},\dotsc\right\}\right)
      =\sum_{n=1}^\infty P\left\{\frac{1}{n}\right\}
      % \ge\sum_{n=1}^\infty\varepsilon 
      \to\infty
    \end{equation}
  #+end_src
  となり矛盾が生じる．
  #+end_quote
  # - 可算個の点からなる事象の確率
  #   #+begin_quote
  #   Lebesgue 測度において
  #   可算個の点からなる事象の確率は
  #   いずれも \(0\) となる．
  #   #+end_quote

#+reveal: split
- 抜き出した点が有理数である確率
  #+begin_quote
  #+begin_src latex
    \begin{align}
      \mathbb{R}_{(0,1]}&=\{\text{区間\((0,1]\)上の実数全体}\}(=\Omega)\\
      \mathbb{Q}_{(0,1]}&=\{\text{区間\((0,1]\)上の有理数全体}\}
    \end{align}
  #+end_src
  と書くことにする．
  有理数は *可算* であるからその要素に番号が付けられ，
  #+begin_src latex
    \begin{equation}
      \mathbb{Q}_{(0,1]}=\{q_1,q_2,q_3,\dotsc,q_n,\dotsc\}
    \end{equation}
  #+end_src
  と書けるので，(P.2)の性質により計算できる．
  #+begin_src latex
    \begin{align}
      P(\mathbb{Q}_{(0,1]})
      &=P\left(\{q_1,q_2,q_3,\dotsc,q_n,\dotsc\}\right)\\
      &=P\{q_1\}+P\{q_2\}+P\{q_3\}+\dotsb+P\{q_n\}+\dotsb\\
      &=0+0+0+\dotsb+0+\dotsb
        =0
    \end{align}
  #+end_src
  #+end_quote

#+reveal: split
- 抜き出した点が無理数である確率
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \text{(区間\((0,1]\)上の無理数全体)}=\mathbb{R}_{(0,1]}-\mathbb{Q}_{(0,1]}
    \end{equation}
  #+end_src
  を用いて求められる．
  #+begin_src latex
    \begin{equation}
      P(\mathbb{R}_{(0,1]}-\mathbb{Q}_{(0,1]})
      =P(\mathbb{R}_{(0,1]})-P(\mathbb{Q}_{(0,1]})=1-0=1
    \end{equation}
  #+end_src
  無理数全体は *可算でない* ため有理数のように成分毎の可算無限和では書けず，
  したがって(P.2)の性質を使って計算することはできない．
  #+end_quote

#+reveal: split
- 異なる区間の場合
  #+begin_quote
  "区間 \((0,5]\) から無作為に一点抜き出す"試行を考えたとき，
  その標本空間は \(\Omega=(0,5]\) となる．
  この試行の確率測度は
  Lebesgue 測度 \(\mu\) を定数倍(正規化)し，
  #+begin_src latex
    \begin{equation}
      P(A)=\frac{\mu(A)}{\mu(\Omega)}=\frac{\mu(A)}{5},\;A\in\mathcal{F}
    \end{equation}
  #+end_src
  とすることによって構成できる．
  #+end_quote

** 零集合
- 定義
  #+begin_quote
  空集合でない集合 \(A\) で
  その事象が起こる確率が \(P(A)=0\) となるものがある．
  こうした集合を *零集合 (null set)*
  と呼ぶ．
  #+end_quote
- 以下は等しくないことに注意 (有理数と無理数の例)
  - 事象 \(A\) の確率が \(0\) である
  - 事象 \(A\) に含まれる標本点が全く起こらない

** "ほとんど確実"
- 確率での特殊な言い回し
  #+begin_quote
  事象
  \(A=\{\omega|\alpha(\omega)\}\)
  の確率が1であるとき，
  *"ほとんど確実に (almost surely)"*
  あるいは
  *"条件 \(\alpha(\omega)\) が確率1で成り立つ"*
  といい，
  以下のように書く．
  #+begin_src latex
    \begin{equation}
      \alpha(\omega)\;\text{a.s.}
    \end{equation}
  #+end_src
  #+end_quote
- 別の表現
  - 事象 \(A\) の余事象 \(A^c\) は零集合
  - 条件 \(\alpha\) は成り立たないこともあるが，その確率は0

  ## 描画領域を指定
  plot(c(0,r),c(0,r),type="n",ann=FALSE,axes=FALSE)

  box()
  
  ## 平行線を引く
  for (z in seq(0,r,by=d)) {
      lines(c(z,z), c(-0.5,r+0.5), col="blue", lty = "solid", lwd = 2)
  }
  
  ## 針を投げる
  for (i in 1:n) {
      x <- runif(1,min=0,max=r)
      y <- runif(1,min=0,max=r)
      t <- runif(1,min=0,max=2*pi)
      x1 <- x-l/2*cos(t)
      x2 <- x+l/2*cos(t)
      y1 <- y-l/2*sin(t)
      y2 <- y+l/2*sin(t)
      lines(c(x1,x2),c(y1,y2),
	    col=ifelse(floor(x1)-floor(x2)==0,"green","red"),
	    lty = "solid", lwd = 2)
  }
  ## 題
  title(main="Buffon's needle")
